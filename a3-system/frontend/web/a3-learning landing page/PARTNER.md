## Technology Partners

### iFlytek

**Role**: LLM and AI Technology Partner

iFlytek provides the Spark LLM foundation that powers A3's intelligence layer:
- **Spark LLM v3.5+**: Core reasoning and natural language understanding
- **TTS SDK**: Voice synthesis for tutoring features
- **ASR**: Speech recognition for voice input
- **Content Moderation API**: Safety filtering for educational content

**Partnership Highlights**:
- Integration with iFlytek's advanced Chinese language models
- WebSocket-based streaming for real-time interactions
- Enterprise-grade content moderation
- Competition track alignment (15th China Software Cup)

**Website**: [open.xfyun.cn](https://open.xfyun.cn)
