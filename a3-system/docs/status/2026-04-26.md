#### Adaptive Path Planning Algorithm (`agents/`)
- [x] Created `agents/path_planner.py`
  - KnowledgeGraph class: loads from JSON, adjacency lists, topological sort
  - StudentState dataclass: encapsulates student attributes
  - AdaptivePathPlanner class with A*-guided topological sort
  - Cost function: f(n) = g(n) + h(n) + lambda1 * profile_bias + lambda2 * preference_bonus
  - Profile bias considers: weak points, goals, knowledge base, learning pace
  - Preference bonus for matching content types
  - Milestone generation with configurable chunk size (delta=15)
  - 5 quality metrics calculation:
    - dependency_satisfaction
    - profile_match
    - difficulty_smoothness
    - weak_point_coverage
    - goal_convergence
  - Tested with sample student: 80 nodes, 6 milestones, ~59 hours total

## Architecture Decisions
1. **Database**: PostgreSQL with SQLAlchemy async support for performance
2. **Migrations**: Alembic for version-controlled schema changes
3. **LLM**: OpenRouter (free tier) as primary, iFlytek as optional fallback
4. **Logging**: structlog for structured, context-rich logging
5. **Config**: Pydantic Settings with environment variable validation
6. **Docker**: Multi-stage builds for minimal image sizes
7. **Path Planning**: A*-guided topological sort for complete DAG coverage
8. **Vector DB**: Weaviate primary, in-memory fallback for dev
9. **Knowledge Graph**: Derived from reference research data (80 keywords, 159 triples)
